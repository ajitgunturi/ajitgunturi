# Hi, I'm Ajit

I build software that respects the people who use it.

Most of my work orbits a few core beliefs: **your data should stay yours**, **the best features don't need a server**, and **complexity should live in the architecture, not in the user's face**.

---

### What I'm building

**[Vael](https://github.com/ajitgunturi/vael)** — A privacy-first family finance app for Indian households. Local-first, encrypted, zero-cost. Families get a shared financial picture — net worth, budgets, goals, projections — without surrendering data to anyone. Built with Flutter, Drift ORM, AES-GCM encryption, and Google Drive as a dumb encrypted pipe. 400+ tests, TDD from day one.

**[Forge](https://github.com/ajitgunturi/forge)** — A CLI that gives AI coding assistants superpowers over your GitHub workflow. One install plugs into Copilot, Claude Code, Codex, and Gemini CLI — PR reviews, discussion analysis, commit coaching. Zero lock-in.

**[MeetMind](https://github.com/ajitgunturi/meet-mind)** — On-device meeting intelligence. Whisper transcription, speaker diarization, and LLM-generated summaries — all running locally on Apple Silicon. No cloud APIs, no data leaving the machine.

**[Shellcraft](https://github.com/ajitgunturi/shellcraft)** — A macOS dev machine bootstrapper with profile-based package selection and non-destructive dotfile management.

### What I care about

- **Privacy as architecture** — not a toggle, not a policy. Encryption boundaries, client-side-only processing, cloud providers that see only noise.
- **Local-first** — apps that work on an airplane, cost nothing to run, and don't depend on someone else's uptime.
- **Deterministic over intelligent** — clear math and rules over black-box predictions. Users make judgments; tools give them accurate numbers.
- **Developer tooling** — making the feedback loop tighter, the workflow smoother, the assistant smarter.
